# Spécification Technique : Portfolio "Agence Digitale Congo" (ADC)

## 1. Objectif du Projet
Transformer le portfolio actuel en une plateforme de services d'agence digitale haut de gamme, ciblant spécifiquement les TPE/PME au **Congo (RDC/RC)**. L'objectif est de maximiser la conversion via des prises de rendez-vous (Calendly) et des contacts directs (WhatsApp).

## 2. Public Cible
- **TPE/PME Congolaises** (Kinshasa, Lubumbashi, Goma, etc.).
- Secteurs : Services (Avocats, Santé, Immobilier), Retail/E-commerce local, Startups.

## 3. Identité Visuelle (Design System)
- **Palette :**
  - Primaire : `#0B132B` (Bleu Nuit Profond - Confiance)
  - Secondaire : `#D4AF37` (Or Métallique - Valeur/Qualité)
  - Accent : `#10B981` (Vert Émeraude - Croissance/Local)
  - Fond : `#F8FAFC` (Gris très clair/blanc cassé)
- **Typographie :**
  - Titres : `Outfit` (sans-serif géométrique, moderne)
  - Corps : `Inter` (lisible, professionnel)
- **Styles :** Modern glassmorphism, ombres portées douces, coins arrondis (16px), animations d'entrée fluides.

## 4. Architecture de l'Information (Sitemap)
1.  **Accueil (Home) :** Proposition de valeur forte, preuve sociale locale, services phares.
2.  **Services :** Présentation sous forme de "Packs" (Vitrine, E-commerce avec Mobile Money, SEO Local).
3.  **Réalisations (Portfolio) :** Études de cas "Problème > Solution > Résultat".
4.  **Processus (Méthodologie) :** Les 4 étapes de travail (Brief, Design, Dév, Lancement).
5.  **Contact :** Page scindée avec Formulaire de devis et calendrier de RDV.

## 5. Fonctionnalités Clés & Intégrations
- **Bouton WhatsApp Flottant :** Toujours visible pour le marché local.
- **Formulaire de Devis Intelligent :** Champ budget, type de projet, localisation (Ville).
- **Intégration Calendly :** Pour les rendez-vous professionnels.
- **Optimisation Mobile First :** Priorité absolue pour les utilisateurs congolais.
- **Composants Réutilisables :** `ServiceCard`, `ProjectCard`, `Testimonial`, `CtaSection`.

## 6. Stack Technique
- **Framework :** React (existant)
- **Styles :** Vanilla CSS + React Bootstrap (existant)
- **Routing :** React Router DOM (existant)
- **Icônes :** React Icons (existant)
- **Animations :** Transition CSS / Framer Motion (envisagé)
